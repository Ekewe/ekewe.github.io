# eksec.github.io

Static portfolio (CTF + projects). Deploy on GitHub Pages.

## Usage
- Edit `index.html`, files under `/ctf` and `/projects`.
- Commit to `main`. Pages serves at `https://eksec.github.io` (once enabled in Settings → Pages).

## Structure
```
/
├─ index.html
├─ assets/style.css
├─ ctf/
│  ├─ index.html
│  └─ sample-thm.html
└─ projects/
   ├─ index.html
   └─ sysmon-lab.html
```
